# Knihovna

Webová aplikace pro správu osobní knihovny. Umožňuje přidávat, sledovat a organizovat knihy podle čtenářského stavu.

## Funkce

### Přidání knihy
- Vyplnění základních údajů: název knihy, autor
- Volitelné pole pro rok vydání
- Výběr stavu čtení při přidání

### Správa knih
- Zobrazení všech knih v přehledné tabulce
- Změna stavu knihy přímo v tabulce pomocí rozbalovacího menu
- Smazání knihy s potvrzovacím dialogem

### Stavy čtení
- **Chci číst** - knihy, které plánujete přečíst
- **Čtu** - knihy, které právě čtete
- **Dočteno** - přečtené knihy

### Sledování postupu
- Automatický přehled počtu dočtených knih
- Zobrazení poměru dočtených knih k celkovému počtu

### Ukládání dat
- Automatické ukládání všech knih do lokálního úložiště prohlížeče
- Data zůstávají zachována i po zavření prohlížeče
- Načtení uložených dat při opětovném otevření aplikace
